# InnoVerse
Forest Fire 
Our project is a Forest Fire Prevention Monitoring System designed for early alert before fire starts. The system simulates real-time forest conditions using four main parameters: temperature, humidity, wind speed, and vegetation dryness. Each parameter is assigned a simple score, and a rule-based logic calculates a total “fire risk” level — Low, Moderate, or High. When the system detects High Risk, it triggers a visual alert box on the dashboard and a simple popup message to notify officials or users that immediate action is needed. The program continuously updates every few seconds, generating new random values for each parameter to simulate real sensor readings. The dashboard displays each parameter in a colored card (green for low, yellow for moderate, red for high), making it easy to understand at a glance. This solution is fully implemented in HTML, CSS, and JavaScript, requires no complex AI or machine learning, and is completely student-friendly, demonstrating a working preventive monitoring system for forests that emphasizes prediction and alerting before a fire occurs, rather than responding after the damage is done.
